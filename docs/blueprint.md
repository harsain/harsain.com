# **App Name**: Harsain's Digital Portfolio

## Core Features:

- Personal Profile Display: Showcase Harsain's profile information, mirroring content from their LinkedIn profile, including professional summary and key skills.
- Project Portfolio: Display projects in a gridded layout similar to the reference design, with images and descriptions. Clicking on a project will display more information.
- Blog Section: Implement a blog section, formatted as a list with article previews (title, summary, image, and publication date).
- Contact Form: Enable users to contact Harsain by completing and submitting a contact form.  Collect their name, email, and message, and deliver to an email.
- Dynamic Content Suggestions: Using provided information on projects or general skill sets, use a tool to suggest changes in the blog content (such as tone or tense), and provide dynamic headline copy to increase viewer engagement. A prompt box to allow user inputting will be added for refinement.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to establish a calming and professional tone, reflecting creativity and thoughtfulness.
- Background color: Very light desaturated lavender (#F5F5FF) to provide a gentle backdrop that doesn't distract from the content.
- Accent color: Deep purple (#4B0082) used sparingly for key interactive elements to draw the user's attention and add depth.
- Body and headline font: 'Inter', a sans-serif font that offers a clean, modern appearance, for body and headline.
- Implement a responsive grid layout, adapting to different screen sizes to maintain visual consistency and usability across devices. The layout should mirror the general structure of the linked website with the portfolio and blog sections clearly defined.
- Utilize simple, minimalist icons for navigation and key actions, ensuring they are consistent with the overall aesthetic. Focus on clarity and readability.
- Incorporate subtle animations for transitions and interactions to enhance user engagement without being intrusive. This includes hover effects on portfolio items and smooth scrolling.
